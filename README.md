
# To-Do React App

This is a simple and modern To-Do List application built with **React**, **TypeScript**, and **Vite**. It allows users to add, view, and manage their daily tasks efficiently with a clean UI.

## Features

- Add and remove tasks
- Mark tasks as completed
- Responsive and fast UI
- Built with React + TypeScript + Vite

## Getting Started

Follow these steps to run the project locally:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Open http://localhost:5173 in your browser
```

## Usage

1. Enter a task in the input field and click "Add".
2. Click on a task to mark it as completed.
3. Remove tasks by clicking the delete icon.

## Project Structure

- `src/` - Main source code (components, styles)
- `public/` - Static assets
- `index.html` - Main HTML file
- `vite.config.ts` - Vite configuration
